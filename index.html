WORKS

Project Features：

  • Hash-based storage of data (Hash Database).
  • Persistence of data even if system crashes.
  • User/Admin login system.
  • Input/Output from command line.
  • Graphic User Interface to enhance user experience.

Project Functions:

  • Online search bar function that will crawl data (text and image) from a specific website (I used bookdepository.com as an example) and store it in user/admin database.
  • Ability to remove and modify data stored using IDs.
  • Log box that keep track of every transaction of database.
  • Search function to pull data from a specific time frame.
  • Send current user's database to any email using username and password provided.
  • Open current user's database.

Admin Access：

  • Ability to check log file that contains all user's transactions in their database.
  • Open any user's database.
  • Delete any user's database.
  • Reconstruct any user's database using log file records.
